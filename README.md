# X\_realtime\_search

<br>

旧ツイッターリアルタイム検索エンジン

<br>

## ベアートークン（必須）

Xの

デベロッパーサイトにて取得して



各自で設定してください。



.envファイル内に記述する箇所があります。

<br>

<br>

```
git clone https://github.com/VioletBlend/x-realtime-search.git
cd x-realtime-search
npm init -y
npm install
node server.js
```
