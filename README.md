# X\_realtime\_search



## ベアートークンはXの

デベロッパーサイトにて取得して



各自で設定してください。



場所は、.envファイル内に記述する箇所があります。

```
git clone https://github.com/VioletBlend/x-realtime-search.git
cd x-realtime-search
npm init -y
npm install
node server.js
```
